# 基于HarmonyOS和SpringBoot的扫码点餐平台app

#### 介绍
手机鸿蒙端的扫码点餐app，大学课设项目

#### 软件架构
采用前后端分离架构，前端为鸿蒙手机app，后端为springboot+mybatis
扫码采用Zxing框架


#### 教程
不懂的看文档以及个人博客：www.abyss.website 
若没有，则是我在准备考研，随缘更新。

#### 注
服务端提供了很多接口，包括用户端与商家端。
由于课设，客户端只做了用户端，因此有许多接口未被调用。
另外，觉得有帮助请为我点亮⭐⭐
